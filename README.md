# Channel Monitor — Comment Ideas Agent

Мониторит Telegram каналы каждые 2 часа.
Когда выходит релевантный пост — присылает тебе уведомление с идеями для комментария.

## Как работает

```
Каждые 2 часа:
  → Проверяет 5 каналов
  → Находит новые посты по ключевым словам
  → Claude генерирует 3 варианта комментария
  → Присылает тебе в личку
  → Ты редактируешь и публикуешь сам
```

## Настройка

### 1. Создай репозиторий на GitHub (приватный)
Загрузи все файлы.

### 2. Добавь секреты (Settings → Secrets → Actions)
| Secret | Value |
|--------|-------|
| `ANTHROPIC_API_KEY` | Твой Claude API ключ |
| `TELEGRAM_BOT_TOKEN` | Токен бота The Pulse |

### 3. Сделай репозиторий публичным
Или используй GitHub Pro для приватных scheduled workflows.

### 4. Напиши боту /start
Открой бота в Telegram и напиши любое сообщение чтобы он мог слать тебе уведомления.

## Настройка каналов

Открой `monitor.py` и измени:
- `TARGET_CHANNELS` — список каналов для мониторинга
- `KEYWORDS` — ключевые слова для фильтрации
- `ADMIN_ID` — твой Telegram ID
